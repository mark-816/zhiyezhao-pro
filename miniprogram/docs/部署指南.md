# 职业照Pro — 部署指南

## 1. 微信云开发环境开通

### 1.1 在微信开发者工具中开通
1. 打开微信开发者工具，加载本项目
2. 点击顶部工具栏的 **「云开发」** 按钮（云朵图标）
3. 在弹出的窗口中点击 **「开通」**
4. 选择环境名称（如：`zhiyezhao-pro`），选择按量计费模式
5. 开通完成后，记录 **环境 ID**

### 1.2 初始化云开发 SDK
在 `app.js` 的 `onLaunch` 方法中，确保已包含云开发初始化代码：

```javascript
onLaunch() {
  wx.cloud.init({
    env: '你的环境ID', // 替换为实际环境ID
    traceUser: true,   // 追踪用户
  })
}
```

> 注意：如果环境ID使用 `wx.cloud.DYNAMIC_CURRENT_ENV` 动态获取，需确保云函数与该前端在同一环境。

## 3. 设置环境变量

云函数需要 **REPLICATE_API_TOKEN** 环境变量才能调用 Replicate API。

### 3.1 获取 Replicate API Token
1. 访问 [https://replicate.com](https://replicate.com) 注册/登录
2. 进入 [API Tokens 页面](https://replicate.com/account/api-tokens)
3. 创建一个 Token（免费用户有有限额度，推荐付费）
4. 复制 Token 值（格式如：`r8_xxxxxxxxxxxxxxxxxxxxxxxx`）

### 3.2 在云函数控制台设置环境变量
**方法一：通过微信开发者工具**
1. 在云开发控制台 → 云函数 → 点击 `generatePortrait`
2. 进入 **「函数配置」** 标签页
3. 在 **「环境变量」** 区域，添加：
   - 变量名：`REPLICATE_API_TOKEN`
   - 变量值：`你的Token`（粘贴刚才复制的值）
4. 点击 **「保存」**

**方法二：通过配置文件**
在 `cloudfunctions/generatePortrait/` 目录下创建或编辑 `.env` 文件（仅本地开发用）：
```bash
REPLICATE_API_TOKEN=r8_xxxxxxxxxxxxxxxxxxxxxxxx
```

> ⚠️ 重要：**不要在客户端代码中传递 API Token**。Token 仅通过云函数环境变量注入，前端代码无法读取。

### 3.3 可选环境变量
| 变量名 | 说明 | 默认值 |
|--------|------|--------|
| `REPLICATE_API_TOKEN` | **必填** Replicate API Token | 无 |
| `REPLICATE_FLUX_VERSION` | FLUX 模型版本（不填则用最新稳定版） | `black-forest-labs/flux-dev` |
| `REPLICATE_FLUX_MODEL` | 预留字段 | - |

## 4. 云数据库集合结构

### 4.1 创建 `portraits` 集合
1. 在云开发控制台 → **「数据库」** → **「集合」**
2. 点击 **「创建集合」**，名称填写 `portraits`
3. 权限设置：建议初始设置为 **「仅创建者可读写」**（安全）
4. 如需要历史查看功能，可添加索引：
   - `openId`（升序）
   - `createdAt`（降序）

### 4.2 集合数据结构
每条记录包含以下字段：

| 字段 | 类型 | 说明 |
|------|------|------|
| `_id` | String | 自动生成的文档ID |
| `generateId` | String | 生成批次ID（如 `zp_1680000000000_abc123`） |
| `openId` | String | 用户微信 OpenID（自动获取） |
| `prompt` | String | 发送给 AI 模型的提示词 |
| `style` | String | 选择的风格：`business` / `casual` / `leisure` / `chinese` |
| `background` | String | 选择的背景：`office` / `solid` / `city` / `nature` |
| `numOutputs` | Number | 生成的图片数量 |
| `imageUrls` | Array<String> | Replicate 返回的图片 URL 列表 |
| `predictedAt` | Date | AI 模型完成时间 |
| `createdAt` | Date | 数据入库时间 |

### 4.3 示例数据
```json
{
  "_id": "abc123def456",
  "generateId": "zp_1680000000000_a1b2c3",
  "openId": "oXXXX_xxxxxxxxxxxxx",
  "prompt": "professional business suit, formal office portrait, clean modern office background...",
  "style": "business",
  "background": "office",
  "numOutputs": 4,
  "imageUrls": [
    "https://replicate.delivery/pbxt/XXXXX/output_0.png",
    "https://replicate.delivery/pbxt/XXXXX/output_1.png",
    "https://replicate.delivery/pbxt/XXXXX/output_2.png",
    "https://replicate.delivery/pbxt/XXXXX/output_3.png"
  ],
  "predictedAt": "2025-01-15T10:30:00Z",
  "createdAt": "2025-01-15T10:30:05Z"
}
```

### 4.4 数据安全建议
- 将 **默认权限** 设为 `仅创建者可读写`
- 如需管理员查看全部数据，可创建安全规则：
```json
{
  "read": "doc.openId == auth.openid || auth.role == 'admin'",
  "write": "doc.openId == auth.openid"
}
```

## 5. 数据流说明

```
用户上传照片 → 选择风格/背景 → 点击「生成」
                                    ↓
                        前端调用 wx.cloud.callFunction({
                          name: 'generatePortrait',
                          data: { style, background, numOutputs: 4 }
                        })
                                    ↓
                        云函数接收参数
                                    ↓
                        构建 prompt（中文风格 → 英文提示词）
                                    ↓
                        调用 Replicate API（FLUX Dev 模型）
                          POST https://api.replicate.com/v1/predictions
                                    ↓
                        轮询等待结果（约4秒）
                                    ↓
                        提取图片 URL 列表
                                    ↓
                        存入云数据库 portraits 集合
                                    ↓
                        返回 { code: 0, data: { generateId, images: [...] } }
                                    ↓
                        前端接收结果 → 展示图片 → 用户保存/分享
```

## 6. 常见问题

### Q: 点击生成后报错 "微信云开发未初始化"
**解决**：检查 `app.js` 中是否调用了 `wx.cloud.init()`

### Q: 报错 "缺少 REPLICATE_API_TOKEN"
**解决**：在云函数控制台「函数配置」中添加环境变量 `REPLICATE_API_TOKEN`

### Q: 报错 "AI 生成超时"
**解决**：Replicate 免费版队列较长，高峰期可能超过30秒，可：
1. 升级为付费账户（优先推荐）
2. 在云函数中增加 `MAX_RETRIES`（已设为60秒）

### Q: 报错 "AI 服务调用失败 (401)"
**解决**：API Token 无效或已过期，重新在 Replicate 官网生成新 Token

### Q: 图片加载失败（`onImageError`）
**解决**：
1. Replicate 返回的 URL 有时效性（通常1小时），确认是否超时
2. 可在云函数中将图片下载后转存到云存储（有需要可扩展此功能）
3. 检查微信小程序是否配置了合法域名（需在控制台添加 `api.replicate.com` 和 `replicate.delivery`）

### Q: 保存到相册失败
**解决**：
1. 检查是否已授权相册权限
2. 网络图片需先 `wx.downloadFile` 到本地临时路径
3. 确保 [downloadFile 合法域名](https://developers.weixin.qq.com/miniprogram/dev/framework/ability/network.html) 已配置 `replicate.delivery`

### Q: 如何在微信小程序后台配置合法域名？
进入微信公众平台 → 开发 → 开发设置 → **服务器域名**：
```
request 合法域名：
  https://api.replicate.com

downloadFile 合法域名：
  https://replicate.delivery
  https://pbxt.replicate.delivery
```
