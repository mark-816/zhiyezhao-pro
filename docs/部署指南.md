# 🚀 职业照Pro · 部署指南

> 微信云开发环境开通 + 云函数上传 + API Key设置

---

## 第一步：开通微信云开发

```
1. 打开微信开发者工具 → 打开职业照Pro项目

2. 顶部菜单 → 云开发
   （如果找不到，点工具栏右边的「云开发」图标）

3. 点击「开通」
   - 环境名称：zhiyezhao-pro（或随便起）
   - 选择「按量付费」模式（免费额度够用）
   - 等待创建完成（约1分钟）

4. 开通后记下「环境ID」
   （后面要用到）
```

## 第二步：创建云数据库集合

```
1. 云开发控制台 → 数据库

2. 创建集合：portraits
   - 集合名：portraits
   - 权限：所有用户可读，仅创建者可写

3. 创建索引（可选，用户量大时再建）
   - 字段：openId，索引类型：normal
```

## 第三步：上传云函数

```
1. 微信开发者工具 → 项目管理器

2. 右键 cloudfunctions/ → 选择「云函数目录」
   （确保 project.config.json 中有 cloudfunctionRoot）

3. 右键 generatePortrait → 「上传并部署」
   （选择「上传所有文件」）

4. 等待上传完成（约30秒）
```

## 第四步：设置环境变量

```
1. 微信开发者工具 → 云开发控制台 → 云函数

2. 点开 generatePortrait → 「配置」

3. 环境变量 → 新增：
   键：REPLICATE_API_TOKEN
   值：r8_你的APIKey（从 replicate.com/account/api-tokens 复制）

4. 点击「保存并重新部署」

5. 验证：在云函数列表中点「测试」→ 
   输入 {"numOutputs": 1} → 调用
   应返回 code: 0 和图片URL
```

## 第五步：测试全流程

```
1. 微信开发者工具 → 点击「预览」

2. 用手机微信扫码

3. 测试流程：
   首页 → 点击「立即生成」→ 上传页
   → 选择风格「商务西装」→ 选择背景「办公室」
   → 点击生成 → 等待10-30秒 → 看到AI生成结果
```

## 常见问题

### Q: 上传云函数后报错 "找不到模块"
A: 右键云函数 → 在终端打开 → 执行 npm install

### Q: 生成后没有图片
A: 检查云函数环境变量 REPLICATE_API_TOKEN 是否设置正确

### Q: 支付提示未开通
A: 需要在微信商户平台开通微信支付，测试阶段可跳过

### Q: 提示 "服务端配置错误"
A: 云函数环境变量未设置，按第四步操作
