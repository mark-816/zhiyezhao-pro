## 二、必须提供的法律文件

### 2.1 《用户协议》（小程序内必配）

核心条款：

```
1. 服务说明
   - 本服务使用AI技术为用户生成职业形象照
   - 生成结果仅作为参考，不代表真实形象

2. 用户授权
   - 用户上传照片即授权平台进行AI处理
   - 平台不会将用户照片用于训练模型（除非获得单独同意）
   - 生成完成后30天内照片将被自动删除

3. 知识产权
   - 生成的照片著作权归用户所有
   - 用户可自由使用、下载、传播生成的照片
   - 平台不得将用户生成内容用于商业宣传（除非获得单独同意）

4. 免责声明
   - AI生成结果可能存在偏差，不代表真实长相
   - 用户不得将生成照片用于欺诈、冒充等违法用途
   - 如因用户违法使用导致纠纷，由用户自行承担

5. 服务终止
   - 用户可随时注销账号、删除所有数据
   - 平台有权对违规用户终止服务
```

### 2.2 《隐私政策》（微信审核必查）

核心条款：

```
1. 收集的信息
   - 微信昵称、头像（用于账号识别）
   - 用户上传的照片（用于AI生成）
   - 设备信息（用于兼容性适配）

2. 信息用途
   - 照片仅用于生成AI形象照
   - 生成完成后照片在30天内删除
   - 不会用于任何其他目的

3. 信息共享
   - 不上传到第三方AI训练平台
   - Replicate API仅做实时推理，不留存数据
   - 不向任何第三方出售用户数据

4. 用户权利
   - 可随时查看、导出、删除自己的数据
   - 可要求平台删除所有历史记录

5. 未成年人
   - 14岁以下儿童使用需监护人同意
   - 不主动收集14岁以下用户数据
```

### 2.3 《AI生成内容标识声明》

```
根据《生成式人工智能服务管理暂行办法》：
- 所有AI生成的图片将在右下角添加「AI生成」水印（小字）
- 小程序首页底部标注：本服务使用AI技术生成内容
- 分享到微信时，分享卡片标注「AI生成」
```

## 三、微信小程序审核合规

### 3.1 类目选择

| 类目 | 说明 |
|:----|:------|
| **首选：摄影-证件照制作** | 最贴切，审核通过率高 |
| 备选：工具-图片处理 | 范围更广但可能要求更多资质 |
| ❌ 避免：AI/人工智能 | 审核最严格，非必要不选 |

### 3.2 审核敏感词（需在代码中避免）

```
❌ 换脸        → ✅ AI形象照 / 风格迁移
❌ 变脸        → ✅ AI美化
❌ 深度伪造     → ✅ AI生成
❌ fake         → ✅ generated
❌ deepfake     → ✅ AI-powered
```

### 3.3 必须展示的内容

```
1. 首页底部必须显示：
   本服务使用AI技术生成，结果仅供参考

2. 结果页必须显示：
   AI生成内容 · 可能存在偏差

3. 用户协议和隐私政策必须在小程序内可查阅
```

## 四、数据安全方案

### 4.1 照片处理流程

```
用户上传 ─→ 前端压缩（≤2MB） ─→ 云函数（临时处理） ─→ Replicate API
                                                                  │
                         用户下载 ←── CDN分发 ←── 生成结果存OSS
                             │
                         30天后自动删除 ←── 定时清理函数
```

### 4.2 数据保留策略

| 数据类型 | 保留期限 | 删除方式 |
|:---------|:--------:|:---------|
| 用户上传原图 | 24小时 | 自动清理脚本 |
| AI生成结果 | 30天 | 定时云函数删除 |
| 用户头像昵称 | 直至用户注销 | 手动删除 |
| 支付记录 | 法律规定5年 | 微信支付侧保存 |

### 4.3 安全措施

```
1. 传输加密：全站 HTTPS
2. 存储加密：阿里云 OSS 服务端加密
3. 访问控制：云函数仅允许用户访问自己的数据
4. 日志审计：记录所有敏感操作
```

## 五、知识产权声明

### 5.1 AI生成内容的版权

```
根据中国《著作权法》和《生成式AI管理办法》：

1. 作品的独创性由AI和用户共同贡献
2. 用户对生成结果享有使用权
3. 用户不能申请著作权登记
4. 用户可将生成结果用于个人/商业用途（非违法场景）
```

### 5.2 用户上传照片的版权

```
用户承诺：
1. 上传照片为用户本人/已获授权
2. 不上传他人照片用于生成
3. 不涉及侵犯他人肖像权、名誉权
```
